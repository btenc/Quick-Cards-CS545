# Quick Cards - CS545

Group members: William TenCate, Ant Vilotti, Drayton Barrand, Jason Scheckner

A fast, accessible, keyboard/mouse optimized flashcard app built with React, TypeScript, and Vite.  
All data is stored locally in your browser or saved on export.

Focused on efficiency over other flashcard systems.

## Features

- Study mode with progress tracking
- Edit mode with autosave
- Keyboard shortcuts (Space, G, ← →, S, D)
- Mouse and touch support
- Responsive UI to fit device width and height.
- System and browser zoom compliant.
- Import / Export decks as JSON.
- Light + Dark themes (or Auto).
- Accessible: Built around HCI and
  accessibility best practices: colorblind-safe palette,
  screen-reader labels, predictable tab and reading order, large click areas
  reduced-motion compliance, high-contrast UI, full zoom
  support.
